|logosub| Mapinou
#################

.. |logosub| image:: doc/lapinou.png
    :height: 25px

**English**

Discover Mapinou, a free mobile game where you have fun while helping science, crazy right?
Download the game on your phone and travel the trails of France to help Lapinou finding his friend.
Beware though! Predators roam the paths as well, and they won't hesitate to attack you if you get lost.

Mapinou is developped by the LostInZoom ERC team, directed by Guillaume Touya. It is a "serious" mobile game
as it uses the videogame format to gather data in order to better understand how we interact with online maps.
The collected data are anonymous.

**Français**

Découvrez Mapinou, un jeu mobile gratuit qui permet de s’amuser tout en aidant la recherche scientifique, incroyable non ?
Téléchargez le jeu sur votre téléphone et parcourez les chemins de France pour aider Lapinou a retrouver son ami.
Mais attention ! Les chemins fourmillent de prédateurs qui n’hésiteront pas à vous croquer si vous vous égarez.

Mapinou est développé par l’équipe du projet européen LostInZoom, dirigé par Guillaume Touya.
C’est un jeu mobile dit « sérieux », c’est-à-dire qu’il utilise le format vidéoludique pour
collecter des données qui vont permettre de mieux comprendre comment nous interagissons avec les cartes en ligne.
Les données collectées sont anonymes.
